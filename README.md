# Buoy Productivity

electron productivity app

## Getting started
1. clone repo
2. enter folder in cmd and run:
3. > npm init
4. > npm install
5. > npm run buoy

## Important Dependencies

- VISUAL STUDIO COMMUNITY 2019 + C++ packages
- > npm i electron-active-window
